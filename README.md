NodeJs & Vagrant Boilerplate
============================

What you get
-------------

* NodeJs    6.0.2
* Npm       3.8.9
* MongoDb   3.0.12

Requirements
------------

* [Vagrant](https://www.vagrantup.com/downloads.html)
* [VirtualBox](https://www.virtualbox.org/wiki/Downloads)
* [RoboMongo](https://robomongo.org/download)

Installation
-------------

* ```vagrant up```
* ```vagrant ssh```
